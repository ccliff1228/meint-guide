# Vim 기본 사용법

## 모드

| 모드 | 진입 | 용도 |
|------|------|------|
| Normal | `Esc` | 이동, 삭제, 복사 |
| Insert | `i`, `a`, `o` | 텍스트 입력 |
| Command | `:` | 저장, 종료, 검색 |

---

## 파일 저장/종료

| 명령 | 설명 |
|------|------|
| `:w` | 저장 |
| `:q` | 종료 |
| `:wq` | 저장 후 종료 |
| `:q!` | 저장 없이 강제 종료 |

---

## 커서 이동 (Normal 모드)

| 키 | 이동 |
|----|------|
| `h` `j` `k` `l` | ← ↓ ↑ → |
| `w` / `b` | 다음/이전 단어 |
| `0` / `$` | 줄 처음/끝 |
| `gg` / `G` | 파일 처음/끝 |
| `:10` | 10번째 줄로 |

---

## Insert 모드 진입

| 키 | 설명 |
|----|------|
| `i` | 커서 앞에 입력 |
| `a` | 커서 뒤에 입력 |
| `o` | 아래에 새 줄 |

---

## 편집 (Normal 모드)

| 명령 | 설명 |
|------|------|
| `x` | 한 글자 삭제 |
| `dd` | 한 줄 삭제 |
| `5dd` | 5줄 삭제 |
| `yy` | 한 줄 복사 |
| `5yy` | 5줄 복사 |
| `p` | 붙여넣기 |
| `u` | 실행 취소 |
| `Ctrl+r` | 다시 실행 |

---

## 여러 줄 선택 (Visual 모드)

| 키 | 설명 |
|----|------|
| `V` | 줄 단위 선택 시작 |
| `j` / `k` | 선택 범위 확장 |
| `y` | 선택 영역 복사 |
| `d` | 선택 영역 삭제 |
| `Esc` | 선택 취소 |

**예시:** `V` → `jjj` (3줄 추가 선택) → `y` (복사) → 원하는 위치로 이동 → `p` (붙여넣기)

---

## 마우스 사용

**기본 (마우스 모드 OFF):**
- 마우스 드래그 → 터미널 클립보드에 복사
- `Ctrl+Shift+V` → 붙여넣기

**Vim 마우스 모드 활성화:**
```vim
:set mouse=a
```
- 마우스 드래그 → Visual 모드 선택
- `y` → 복사, `d` → 삭제, `p` → 붙여넣기

|------|------|
| `/pattern` | 검색 |
